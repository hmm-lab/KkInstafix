# kkInstaFixBot

This bot rewrites supported social media URLs into Telegram-friendly fixer URLs.

## Commands

- /providers
- /setprovider instagram ddinstagram
- /setprovider twitter fxtwitter
- /setprovider x fixupx
- /resetproviders
- /help

## Notes

- Upload these files directly to GitHub instead of copy-pasting code from chat.
- Provider settings are saved per chat.
- The bot rewrites supported URLs and removes common tracking parameters.
- YouTube Shorts are converted to normal watch links.
- If the bot cannot delete the original message, it replies instead.
